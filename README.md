# GitHub Explorer

A GitHub user and repository search application built with React.

## Features

- Search GitHub Users
- Browse Repositories
- Bookmark Repos
- Smart Filtering
- Language Filter
- Theme Toggle
- Pagination
- Responsive Design
- Real-time Stats

## Quick Start

### Installation

clone the repository and navigate to the project directory: cd into the project directory and install dependencies:
```bash
npm install
npm run dev
